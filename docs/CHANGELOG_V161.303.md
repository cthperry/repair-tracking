## V161.303 - 2026-03-13

### 維修管理詳情頁「總覽」企業級重設計
- 將維修詳情頁 `總覽` 從舊式左資訊卡 / 右輔助卡拼接模式，重構為正式的 overview board。
- 新增四個固定總覽區塊：`案件總覽`、`客戶與聯絡資訊`、`設備與機台資訊`、`收費與下單追蹤`。
- 將案件狀態、優先級、負責人、建立/更新時間、維修天數、進度拉回同一層資訊架構，避免資訊散落多區。
- 將客戶 / 電話 / Email 與機台 / 產品線 / 序號拆成獨立資訊卡，改善閱讀順序與企業級可判讀性。
- 將問題描述卡改為 `問題描述與處理背景`，並把內部備註收斂進同一區塊，避免總覽資訊破碎。
- 移除右側重複的收費 / 下單狀態與其他資訊卡，降低重複資訊與實驗性排版感。
- 版號推進至 `V161.303`，`BUILD_NUMBER = 303`。
