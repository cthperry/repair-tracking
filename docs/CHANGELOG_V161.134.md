## 修正

### 1) 新增維修單：設備名稱欄位位置在選擇產品線後會「跑掉」
- 根因：設備名稱 label 內的「篩選」按鈕在清單模式才顯示，使用 `display: none/''` 切換會觸發 reflow，導致 label 高度變化與輸入框上下位移。
- 修正：
  - JS 改用 `visibility` 隱藏/顯示，保留版面空間，避免欄位跳動。
  - 增加 CSS：讓篩選按鈕靠右（`margin-left:auto`），降低換行機率。
