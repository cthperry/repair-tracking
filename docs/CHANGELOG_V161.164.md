# CHANGELOG｜V161.164

日期：2026-01-09

## 功能/改善

- 零件追蹤（以維修單為依據）支援「同一維修單多筆」新增/編輯：
  - 新增「編輯用料/更換（多筆）」批次編輯介面，可在同一視窗一次新增/編輯多筆用料追蹤。
  - 於零件追蹤列表按「新增」或任一項目「編輯」時，改開啟批次編輯器：
    - 可直接選擇維修單並一次維護多筆。
    - 支援「＋ 新增一項」快速新增多列。
    - 支援刪除列（既有資料採軟刪除）。
  - 儲存時自動忽略空白列；若列有填值但缺「零件名稱」會提示並阻止儲存。
