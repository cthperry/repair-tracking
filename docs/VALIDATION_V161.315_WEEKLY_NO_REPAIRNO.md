# VALIDATION V161.315 - Weekly 不顯示維修單號

## 驗證目標
- Weekly 預覽與寄送內容中的案件標題，不得顯示維修單號。
- Weekly 預覽與寄送內容中的案件標題，不得顯示案件 ID。
- 當客戶與機台皆空白時，案件標題應顯示 `未命名案件`。

## 驗證方式
1. 新增單元測試 `tests/unit/weekly.no-repairno.spec.js`。
2. 執行 `npm test`。

## 預期結果
- 含有 `repairNo` 的資料不會出現在週報案件標題。
- 含有 `id` 的資料不會出現在週報案件標題。
- 空白案件標題會顯示 `未命名案件`。
