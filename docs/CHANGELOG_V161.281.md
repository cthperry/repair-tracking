## V161.288
- Firebase Auth 新增持久化設定：初始化時自動套用 `AppConfig.auth.persistence`，預設為 `local`，支援手機 / 瀏覽器關閉後保持登入
- 補上初始 auth 狀態等待流程：啟動時先等待 `onAuthStateChanged` 首次回傳，再決定是否進入系統或顯示登入頁，降低登入狀態 race condition
- 關閉預設本地模式降級：`allowLocalFallback` 預設為 `false`，避免網路異常時誤切本地登入造成未授權進入風險
