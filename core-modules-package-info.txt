🚀 使用方式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 解壓縮檔案到專案根目錄
   tar -xzf core-modules-complete.tar.gz
   或
   unzip core-modules-complete.zip

2. 在 HTML 中引入模組（按順序）：

   <!-- CSS 模組 -->
   <link rel="stylesheet" href="core/variables.css">
   <link rel="stylesheet" href="core/shared.css">

   <!-- JavaScript 模組 -->
   <script src="core/constants.js"></script>
   <script src="core/router.js"></script>
   <script src="core/app.js"></script>

3. 初始化應用程式：

   Desktop 版本：
   initializeApp(DesktopApp);

   Mobile 版本：
   initializeApp(MobileApp, {
     constantOverrides: {
       HEADER_TITLE_ID: 'mobile-header-title',
       HEADER_SUBTITLE_ID: 'mobile-header-subtitle'
     }
   });
